# AI Engineering Assistant

A React-based web application for engineers to interact with AI for document processing and technical Q&A.

## Features

- 🤖 AI-powered chat interface
- 📄 Document upload and analysis
- 🔐 AWS Cognito authentication
- 📱 Responsive design
- 🚀 AWS Amplify deployment

## Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn
- AWS Account
